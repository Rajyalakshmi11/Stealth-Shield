# Stealth Shield

## Overview

Stealth Shield is a Python-based application that uses image steganography techniques to securely hide and retrieve secret messages within digital images. The project provides a user-friendly graphical interface that allows users to encode confidential information into images and decode it when required.

## Features

* Hide secret text messages inside images.
* Retrieve hidden messages from encoded images.
* Simple and interactive graphical user interface.
* Security key-based message protection.
* Supports common image formats.
* Lightweight and easy to use.

## Technologies Used

* Python
* Tkinter
* Pillow (PIL)
* Image Steganography

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Stealth-Shield.git
```

2. Navigate to the project directory:

```bash
cd Stealth-Shield
```

3. Install the required dependency:

```bash
pip install pillow
```

4. Run the application:

```bash
python code2.py
```

## Usage

### Encoding a Message

1. Select an image.
2. Enter the secret message.
3. Provide a security key.
4. Save the encoded image.

### Decoding a Message

1. Open the encoded image.
2. Enter the correct security key.
3. View the hidden message.
